Royal DS

Um Design System completo e moderno, desenvolvido para aplicações cross-platform. Baseado em tokens semânticos que garantem consistência visual e facilidade de manutenção em projetos React e Flutter.

Princípios do Sistema

Tokens Semânticos

Todos os tokens referenciam primitivos usando notação de chaves, permitindo mudanças cascateadas em todo o sistema.

Cross-Platform

Estrutura JSON unificada que pode ser consumida por React e Flutter, garantindo consistência entre plataformas.

Contexto Adaptável

Suporte para variações de contexto (Bank/Pro) e dispositivo (Mobile/Tablet/Desktop).

Colors

Sistema completo de cores com primitivos e tokens semânticos para surface, border, text e icon.

Typography

Sistema tipográfico usando JUST Sans Variable com pesos específicos (Light 200, Regular 350, Bold 550).

Spacing

Sistema de espaçamento com primitivos e tokens semânticos para inset, inline e stack.

Grid

Sistema de grid responsivo com definições para mobile, tablet e desktop.

Shadows

Tokens de sombra com controle de distância, blur e spread.

Borders

Tokens para border radius e width.

Opacity

Escala de opacidade para transparências e overlays.

Sizes

Tokens de tamanho para ícones, escalas, divisores e thumbs.
